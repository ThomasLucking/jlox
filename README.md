# jlox — Crafting Interpreters

Working through Robert Nystrom's [Crafting Interpreters](https://craftinginterpreters.com) building a tree-walk interpreter for the Lox language in Java.

## Progress
- [x] Chapter 4 — Scanning / Lexical Analysis

## Running
Open in IntelliJ and run `Lox.java`. You'll get a REPL prompt where you can type Lox code and see the tokens it produces.
